## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Get a free API Key at [https://disease.sh/](https://disease.sh/)
2. Clone the repo
   ```sh
   git clone https://github.com/cucer/covid19.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
